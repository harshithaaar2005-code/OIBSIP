# Oasis Infobyte Data Science Internship - Task 2

## Project: Customer Segmentation using K-Means Clustering
This project involves performing Exploratory Data Analysis (EDA) and applying Machine Learning to segment retail customers based on their purchasing behavior.

### Key Features:
- **Data Cleaning:** Handled missing values and date formatting.
- **EDA:** Visualized spending trends and customer demographics using Seaborn.
- **Machine Learning:** Implemented K-Means Clustering with Scikit-Learn.
- **Optimization:** Used the Elbow Method to find the optimal number of clusters.

### Tech Stack:
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Scikit-Learn (KMeans, StandardScaler)
